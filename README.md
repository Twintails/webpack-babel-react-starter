# Webpack, Babel, React, Starter-Pack

## Reqs
1. `node > 8`
1. `npm`

## Install

```bash
npm i
```
## Run

| Command         | Does This                       |
| --------------- | :-----------------------------: |
| `npm start`     | runs the dev webpack-dev-server |
| `npm run dev`   | build with dev mode into dist   |
| `npm run build` | build with prod mode into dist  |
